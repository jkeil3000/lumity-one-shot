# Lumity One Shot

Desktop-first Lumity prototype built with React, TypeScript, Vite, Tailwind v4, and mock data only.

## Run

```bash
npm install
npm run dev
```

The app runs on `http://localhost:5173` by default.

## Build

```bash
npm run build
```

## Current Scope

- Pages: Home, Explore, Stream, Mind, Profile
- State: in-memory only via React context
- Data: `src/data/mock.ts`
- Styling: token-driven Tailwind classes via `src/index.css`

See `CLAUDE.md` for product, design, and implementation guidance.
